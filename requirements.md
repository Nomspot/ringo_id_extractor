# Requirements

## Installation

Install all dependencies:

```bash
pip install -r requirements.txt
```

---

## Core Libraries

* **selenium** – browser automation for scraping chat data
* **undetected-chromedriver** – bypass detection mechanisms
* **pandas** – data processing and Excel handling
* **requests** – HTTP requests
* **tqdm** – progress bars
* **tenacity** – retry logic for stability
* **pillow** – image processing

---

## Google Cloud / AI

* **google-generativeai** – Gemini model access
* **vertexai** – Vertex AI integration
* **google-cloud-storage** – file handling in cloud
* **google-api-core** – API support

---

## Notes

* Tested on python 3.14
* Google Cloud credentials must be configured
* Chrome browser required for Selenium
